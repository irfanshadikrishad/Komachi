#### Komachi

##### An anime streaming platform to watch and download your favourite anime's online.

#### features:

- No ads.
- Comment functionalty through [disqus](https://disqus.com).
- Clean UI.
- Fast anime updates.
- Multiple stream source.
- Auto Skip Intro, Outro
- Auto Play next episode

#### tech stacks:

- ExpressJS
- NodeJS
- ReactJS
- Bun
